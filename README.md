Dependencies  
  * node v16.10.0

&nbsp;

Setup:
npm install  

Android  
  * Emulator: genymotion  

  * Run app
    - npx react-native start  
    - Start emulator
    - npx react-native run-android  

iOS
  * Emulator: Xcode  

  * Run app
    - npx react-native start  
    - Start emulator  
    - npx react-native run-ios
